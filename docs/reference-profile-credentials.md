# Эталонный профиль для тестирования

Профиль для синхронизации с заказчиком — оба тестируют с одного аккаунта.

---

## Демо-сборка (моки, без бэкенда)

**URL:** https://angelo-test.ru (после `deploy-demo.sh`)

**Вход:**
1. Открыть https://angelo-test.ru/demo
2. Нажать «Войти в демо»

**Креды не нужны** — демо включается локально в браузере.

**Профиль:**
- **Имя:** Алина Фадеева (Мама)
- **Семья:** Дедушка, Бабушка, Папа, Мама, Лиза + 45 участников (в стресс-режиме)
- **Публикации:** 200+ (в стресс-режиме)

---

## Реальный API (бэкенд)

**URL:** https://angelo-test.ru (после `deploy-fornex.sh`)

**Эталонный идентификатор:**
```
alina.fadeeva@angelo-demo.ru
```

**Код:** любой 4–6 цифр (бэкенд пока не проверяет код), например `000000`.

**Шаги:**
1. Ввести `alina.fadeeva@angelo-demo.ru`
2. Нажать «Получить код»
3. Ввести `000000` (или любой код)
4. Нажать «Подтвердить»

**Важно:** первый, кто войдёт с этим email, создаст аккаунт. Дальше все, кто вводит тот же email и любой код, попадают в этот же профиль. Для наполнения семьи и публикаций нужен seed или ручное добавление данных.
