## Нужно сделать (приоритет)

1. **Авторизация** — бэкенд + отправка кода (SMS/email или Telegram).
2. **Фильтры ленты** — автор, участники, даты, избранное, непрочитанное.
3. **Семья** — вкладки Обо мне / Семья, Профили / Группы.
4. **Профиль контакта** — блоки «Публикации» и «Медиа» с переходами.
5. **Настройки** — имя, фото, «Выйти», условия/политика.
6. **Приглашения** — принять/отклонить, копировать/отменить.
7. **Экран «Места»** — список, пригласить на место, освободить.
8. **Видимость публикации** — выбор «видна для» / «исключить».
