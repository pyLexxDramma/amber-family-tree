# Angelo — экраны для презентации клиенту

Скриншоты сохраняются в:
- `docs/screenshots/` — десктоп (1280×720)
- `docs/screenshots-mobile/` — смартфон (390×844, iPhone)

Запуск: `npm run screenshots` (dev-сервер должен быть запущен на localhost:8080).

## Дополнительные возможности

- **Голосовой помощник** — кнопка с микрофоном в правом нижнем углу. Одно нажатие — переход в Angelo с приветствием.
- **Журнал + Плеер** — режим для слабовидящих: озвучка публикаций голосом.
- **Демо** — просмотр без регистрации по ссылке «Демо» на главной.
