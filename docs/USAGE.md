# Инструкция по использованию Angelo

Angelo — семейный альбом для хранения воспоминаний, общения с близкими и просмотра генеалогического древа.

---

## Начало работы

**Демо-режим** — на главной нажмите **«Демо»**. Откроется лента, дерево семьи и профили с фото. Без регистрации.

**Вход / Регистрация** — введите телефон или email → код подтверждения → онбординг (заполните профиль) или «Пропустить».

---

## Выбор интерфейса

**Настройки** → **Вариант оформления**. Выберите один из пяти режимов (применяется сразу):

| Вариант | Описание |
|---------|----------|
| **Текущий** | Как выглядит приложение сейчас |
| **Классический семейный архив** | Обложка журнала, витрина ленты альбомов |
| **Живая история** | Разворот с мемуарами, фото с подписью, навигация по публикациям |
| **Календарь воспоминаний** | Интерактивная хронология, лента по годам |
| **Журнал + Плеер** | Озвучка для слабовидящих, просмотр с аудио |

---

## Журнал + Плеер (для слабовидящих)

В режиме **Журнал + Плеер** публикации озвучиваются голосом. Откройте любую публикацию в ленте:

- Кнопка **«СЛУШАТЬ ИСТОРИЮ»** — запускает озвучку текста публикации
- Внизу — большая кнопка Play/Pause для включения и остановки
- Текст подписи читается вслух синтетическим голосом
- Можно сохранить фото в телефон или перейти к другой публикации

Путь: **Настройки** → **Вариант оформления** → **Журнал + Плеер**.

---

## Основные разделы

**Дерево** — семейное древо по поколениям. Карточка участника → профиль с фото, родителями, детьми.

**Лента** — публикации семьи (фото, видео, аудио, текст). Фильтры: автор, участники, даты, темы, избранное. Вкладки «Публикации» и «Медиа».

**Семья** — список членов семьи. Табы: Все / Активные / Неактивные. «Обо мне» — ваш профиль. «Пригласить» — приглашение родственников.

**Создать** (кнопка + внизу) — новая публикация.

**Магазин** — тарифы и подписки.

---

## Создание публикации

1. Нажмите **+** в нижнем меню → **Создать публикацию**
2. Выберите тип: **Photo**, **Video**, **Audio**, **Document**, **Text**
3. Заполните:
   - Заголовок (необязательно)
   - Описание
   - Дата события (можно пометить «приблизительно»)
   - Место
   - **Тема*** (обязательно): Праздники, День рождения, Свадьба, Будни, Путешествия, Рецепты, Истории, Детство
4. Приложите файлы (кнопка «Добавить файл»)
5. Настройте видимость (по умолчанию — всем членам семьи)
6. Нажмите **Опубликовать**

---

## Загрузка фото и медиа

- **Фото профиля:** онбординг или Мой профиль → Редактировать профиль (кнопка с камерой)
- **В публикации:** Создать → выбрать Photo/Video/Audio → Добавить файл
- **Форматы:** фото JPG/PNG/WebP до 20 МБ, видео MP4 до 500 МБ, аудио M4A/MP3 до 100 МБ

## Голосовой помощник

Кнопка с микрофоном в правом нижнем углу. Нажмите и скажите команду:

| Команда | Действие |
|---------|----------|
| «Дерево», «Покажи дерево» | Дерево семьи |
| «Лента», «Что нового» | Лента |
| «Семья», «Контакты» | Список семьи |
| «Расскажи про дедушку» | Открыть помощника, рассказать про человека |
| «Настройки», «Помощь» | Перейти в раздел |
| «Назад» | Назад |

В помощнике можно писать или говорить: «Покажи дерево», «Расскажи про Ольгу», «Что нового?», «Покажи фото».

---

## Профиль и настройки

**Мой профиль** — вкладка «Семья» → «Обо мне» (или в медиа-режиме ленты — вкладка «Профиль»). Редактирование: Редактировать профиль.

**Профиль контакта** — дерево или семья → карточка участника. Фото, даты, родители, дети.

**Настройки** — Мой профиль → Настройки. Доступно: тема (светлая/тёмная), уведомления, приватность (всем / семья / только мне), язык (RU/EN), варианты оформления, условия, политика, помощь.

---

## Поддерживаемые форматы

- **Фото:** JPG, PNG, WebP (до 20 МБ)
- **Видео:** MP4 (до 500 МБ)
- **Аудио:** M4A, MP3 (до 100 МБ)

## Частые вопросы

**Как пригласить родственников?**  
Семья → Пригласить → отправить ссылку.

**Как работает голосовой помощник?**  
Распознаёт речь, отвечает на вопросы о семье, показывает дерево, ленту, галерею, подсказывает создание публикации.

**Как включить режим для слабовидящих?**  
Настройки → Вариант оформления → Журнал + Плеер.

**Можно ли выгрузить данные?**  
Настройки → Экспорт данных — фото, истории и дерево архивом ZIP.

**Данные конфиденциальны?**  
Да. Контент виден только приглашённым членам семьи.
