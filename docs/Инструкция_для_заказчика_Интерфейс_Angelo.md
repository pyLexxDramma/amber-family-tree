## Инструкция по интерфейсу Angelo (для заказчика)

Актуально для мобильного интерфейса в разделе `classic`.

### 1) Нижняя навигация

Внизу экрана закреплено 5 разделов:

- **Лента**: все публикации семьи и режим «медиа-сетка».
- **Таймлайн**: временная шкала событий семьи с фильтрами.
- **Создать**: добавление новой публикации (история/фото/медиа).
- **Семья**: список членов семьи, переход в профили.
- **Дерево**: семейное древо, связи и фокус на человеке.

Нажатие на пункт открывает соответствующий экран. Активный пункт подсвечивается.

### 2) Лента

Экран предназначен для просмотра всех публикаций семьи.

- **Вкладки «Лента / Посты»**:
  - **Лента**: визуальный просмотр (в том числе «медиа-сетка»).
  - **Посты**: список публикаций карточками, где видно автора, дату, заголовок и текст.
- **Сетка (1 / 3 / 5)**: влияет на плотность отображения медиа в «Ленте». Чем больше число — тем мельче плитки.
- **Сначала новые / Сначала старые**: сортировка по дате публикации.
- **Фильтры**: быстрый выбор публикаций (по автору/участию и другим условиям, если включены).
- **Открытие публикации**: нажатие по карточке/плитке открывает страницу публикации с деталями.

Что показывается в карточке публикации:

- **Автор** (имя).
- **Дата события** (если указана) и/или дата публикации.
- **Заголовок** и **текст** (в списке может быть сокращён до нескольких строк).
- **Место** (если указано).
- **Медиа** (фото/видео/аудио — в зависимости от типа публикации).

### 3) Публикация (детальный экран)

На детальной странице доступны:

- просмотр медиа (фото/видео и т.д.);
- заголовок, текст, дата события, место;
- лайки и комментарии;
- участники публикации (если заданы).

### 4) Создать (добавление публикации)

Раздел используется для создания нового контента.

Обычно шаги такие:

- выбрать тип публикации (например, фото/история/альбом — в зависимости от доступных вариантов);
- добавить медиа (если требуется);
- заполнить **заголовок** и **текст**;
- указать **дату события** и (опционально) **место**;
- при необходимости выбрать **тему/тег** и **участников**;
- опубликовать.

### 5) Семья (список членов семьи)

Экран показывает перечень участников семьи.

- сверху доступен поиск/фильтрация (если включено);
- в каждой строке: **аватар**, **имя**, **роль/родство** (например: мама, папа, дедушка);
- нажатие на участника открывает его профиль.

### 6) Профиль участника

В профиле отображаются данные человека (имя, дата рождения, город, описание), а также связанные материалы:

- публикации автора;
- публикации с его участием (если поддерживается);
- медиа (если включено).

### 7) Дерево (семейное древо)

Экран «Дерево» помогает быстро понимать структуру родства.

- в центре — **фокусный человек** и ближайшие связи;
- можно переходить по родственникам, меняя фокус;
- в блоке связей/вкладках доступны переходы в профиль, истории/публикации и медиа.

Связи строятся на основе родственных отношений (родитель/ребёнок/супруг/брат-сестра).

## 8) Таймлайн (подробно)

**Таймлайн** — это экран, где публикации и события семьи показываются по времени. Он нужен, чтобы быстро найти воспоминания за конкретный год/период и сравнивать периоды между собой.

### 8.1 Как формируется таймлайн

Таймлайн группирует контент по **дате события** (если она задана в публикации). Если дата события отсутствует, используется дата публикации.

На таймлайне вы видите:

- **периоды** (например, годы/десятилетия — в зависимости от режима);
- **количество событий** в периоде;
- **обложку** (превью одного из событий периода).

Нажатие на период открывает список событий внутри него.

### 8.2 Режимы просмотра

В таймлайне обычно используются уровни детализации:

- **Десятилетия**: быстрый обзор по 10-летним периодам.
- **Годы**: список лет с количеством событий.
- **Месяцы/события**: более детальный просмотр внутри выбранного года.

Если в интерфейсе доступен переключатель уровня (например, «десятилетия → годы → месяцы»), он меняет масштаб времени.

### 8.3 Фильтры таймлайна

Фильтры помогают быстро «сузить» события:

- **Люди**: показывать события конкретного автора или события с участием выбранного человека.
- **Места**: фильтр по городу/локации, указанной в публикации.
- **Теги**: фильтр по теме/тегу публикации.
- **Типы**: фото/видео/аудио/история (в зависимости от доступных типов).

Фильтры можно комбинировать (например, «Люди + Места»).

### 8.4 Переход из таймлайна в публикацию

Когда вы открываете события конкретного периода, каждое событие ведёт в:

- детальную страницу публикации (медиа + текст + дата/место), либо
- экран периода/года (если это промежуточный уровень).

### 8.5 Практические сценарии

- **Найти воспоминания за определённый год**: Таймлайн → выбрать год → открыть нужное событие.
- **Найти события с конкретным человеком**: Таймлайн → Фильтры → Люди → выбрать человека → открыть период/событие.
- **Найти события из конкретного города**: Таймлайн → Фильтры → Места → выбрать город → открыть событие.
- **Собрать подборку на праздник**: Таймлайн → Теги/Типы → отфильтровать → открыть нужные публикации.

### 8.6 Что важно учитывать

- **Корректность таймлайна зависит от дат**: чтобы событие оказалось в нужном году, в публикации должна быть указана правильная дата события.
- **Одна публикация = одно событие** (если не используется альбом/серия, где внутри несколько элементов медиа).
